A custom plugin to fetch NFL team data from api end list within shortcode

== Description ==

You will need api key.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `nfl-wp.zip` from `/wp-admin/plugin-install.php`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit 'NFL WP' page and configure the api key. 
4. Choose & generate shortcode from 'NFL WP' page and use it in your website
